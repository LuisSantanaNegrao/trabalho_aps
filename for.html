O que é o laço for?

O laço for é ideal para situações em que sabemos exatamente quantas vezes o bloco de código deve ser executado. Ele segue a seguinte estrutura:

for (inicialização; condição; incremento) {
    // Código a ser executado
}

Exemplo 1: Contar de 1 a 10

for (let i = 1; i <= 10; i++) {
    console.log(i);
}

Exemplo 2: Tabuada de 5

for (let i = 1; i <= 10; i++) {
    console.log("5 x " + i + " = " + (5 * i));
}
