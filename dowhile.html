O que é o laço do...while?

O laço do...while executa o bloco de código pelo menos uma vez, antes de verificar a condição.

do {
    // Código a ser executado
} while (condição);

Exemplo 1: Gerar números aleatórios até obter um valor maior que 5

let numero;
 
do {
    numero = Math.floor(Math.random() * 10);
    console.log("Número gerado: " + numero);
} while (numero <= 5);
