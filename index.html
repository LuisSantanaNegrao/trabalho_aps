Introdução

As estruturas de repetição permitem executar um bloco de código várias vezes, até que uma condição seja atingida. Elas são essenciais para automatizar tarefas repetitivas em algoritmos.

Estrutura for

A estrutura for é utilizada quando o número de iterações é conhecido. Sua sintaxe básica é:

for (inicialização; condição; incremento) {
    // Código a ser executado
}

Exemplo: Listar os números de 1 a 10.

for (let i = 1; i <= 10; i++) {
    console.log(i);
}

Estrutura while

A estrutura while é usada quando não sabemos ao certo o número de iterações, mas sabemos a condição de parada. Sua sintaxe básica é:

while (condição) {
    // Código a ser executado
}

Exemplo: Calcular a soma dos números até que o total seja maior que 20.

let soma = 0;
let num = 1;
 
while (soma <= 20) {
    soma += num;
    num++;
    console.log("Soma atual: " + soma);
}

Estrutura do...while

A estrutura do...while é semelhante ao while, mas ela garante que o bloco de código será executado pelo menos uma vez antes da verificação da condição. Sua sintaxe básica é:

do {
    // Código a ser executado
} while (condição);

Exemplo: Continuar pedindo um valor aleatório até que ele seja maior que 5.

let numero;
 
do {
    numero = Math.floor(Math.random() * 10);
    console.log("Número gerado: " + numero);
} while (numero <= 5);
